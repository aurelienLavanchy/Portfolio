Projet 2: HomelyFood

Première réalisation / utilisation d'une API

Détails

Ce deuxième projet a porté sur l'utilisation d'une API, initialement sur l'exploitation d'une API publique.

Ne trouvant pas d'API publique qui répondait à nos besoins, nous avons créé notre propre API.

Cette dernière n'a pas d'accès à une base de données, les données proviennent directement d'un fichier JSON (car nous n'avions pas encore vu les bases de données à ce stade).

Durée: ~1 mois, du 03/04/2024 au 30/04/2024

Repo: Github

Déploiement: Vercel
